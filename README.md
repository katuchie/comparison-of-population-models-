# comparison-of-population-models-

Проект реализует три математические модели роста популяции и проводит их сравнительный анализ на реальных демографических данных:

1. **Модель Мальтуса** (Malthus) — экспоненциальный рост
2. **Модель Ферхюльста** (Verhulst) — логистический рост с насыщением
3. **Модель Арнольда** (Arnold) — модифицированная логистическая модель
